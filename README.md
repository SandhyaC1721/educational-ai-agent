# AI Study Assistant

## Project Description
AI Study Assistant is a web-based application that helps students study effectively by generating quizzes and providing study recommendations based on uploaded or pasted study material.

## Features
- Upload PDF or TXT study material
- Paste study text manually
- Select subject and difficulty level
- Generate quiz questions
- Display study recommendations

## Technologies Used
- Python
- Streamlit
- PyPDF2

## Project Timeline
- Week 1: Project setup
- Week 2: User interface design
- Week 3: Quiz generation
- Week 4: File handling
- Week 5: Logic enhancement
- Week 6: Feature improvement
- Week 7: UI enhancement
- Week 8: Final integration and testing

## How to Run
1. Install required packages:
   pip install streamlit PyPDF2
2. Run the app:
   streamlit run app.py

## Outcome
A complete AI Study Assistant application that allows students to practice quizzes and follow study recommendations.
